# Guardianes AntiLlama

Aplicación web educativa interactiva para sensibilizar y educar a niños de 9 a 12 años sobre la prevención de quemaduras.

## Características principales

- **Historias interactivas** basadas en casos reales anonimizados
- **Sistema de decisiones** con evaluación pedagógica
- **Motor de IA controlado** con plantillas validadas
- **Indicadores de comprensión** y prevención
- **Reporte final personalizado** para cada niño
- **Accesibilidad completa** con audio y navegación simplificada

## Principios éticos

- ❌ No datos personales
- ❌ No login requerido
- ❌ No imágenes realistas de lesiones
- ❌ No contenido violento o traumático
- ✅ Enfoque en prevención y aprendizaje
- ✅ Empatía y claridad pedagógica
- ✅ Diseño accesible para todos

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Construcción

```bash
npm run build
npm start
```

## Stack tecnológico

- **Frontend**: Next.js 14 + React 18 + TypeScript
- **Estilos**: TailwindCSS
- **Iconos**: Lucide React
- **Motor de historias**: Sistema de plantillas controladas
- **Evaluación**: Modelo de scoring con indicadores

## Estructura del proyecto

```
/app                 # Páginas y rutas de Next.js
/components          # Componentes React reutilizables
/lib                 # Lógica de negocio y utilidades
  /stories           # Motor de historias
  /evaluation        # Sistema de evaluación
  /ml                # Modelo de scoring
/public              # Archivos estáticos
```

## Público objetivo

Niños y niñas de 9 a 12 años en contexto latinoamericano (Perú), con nivel de lectura variable y acceso a dispositivos de gama media o baja.

## Mensaje central

**No enseñamos a tener miedo al fuego. Enseñamos a cuidarse para la vida real.**
